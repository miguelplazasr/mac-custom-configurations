# mac-custom-configurations

## Brew

-Install
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## iTerm

```bash
brew install iterm2
```

## zsh

```bash
brew install zsh
```

